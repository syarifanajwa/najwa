Komponen HTML

Berikut beberapa komponen utama HTML yang umum digunakan dalam pembuatan halaman web.

Heading

Elemen heading digunakan untuk menampilkan judul atau heading dengan tingkat hierarki yang berbeda.

Paragraph

Element paragraph (`p`) digunakan untuk menampilkan paragraf teks.

Hyperlink

digunakan untuk membuat tautan (hyperlink) ke halaman web lain atau sumber daya lainnya.
